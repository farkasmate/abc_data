# AbcData

Format data into alphabetical order.

*Mostly to diff them.*

## Installation

Add this line to your application's Gemfile:

```
gem 'abc_data'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install abc_data
```

## Usage

```
$ abc_diff <a.json> <b.json> [diffParams]
```

### Example

```
$ abc_diff old.json new.json --unified

```
